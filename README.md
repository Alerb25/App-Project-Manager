# Content Manager
Proyecto de clase de JavaFX, es una app para gestionar contenido

Mi proyecto consiste en una aplicación que será capaz de administrar y guardar proyectos,  un proyecto consta de varios atributos (un nombre, una descripción, una imagen y un enlace a github o similares.) La aplicación deberá permitir a un usuario autenticado editar sus propios proyectos, crear nuevos o incluso eliminarlos, y también podrá compartir proyectos con otros usuarios.
